Справочная таблица: Атрибуты требований
Данная справочная таблица позволяет проверить правильность определения требования.
Взаимосвязи
Связанные элементы
  • Атрибуты требований
  • Проверка требований
Основное описание
Элементы справочной таблицы
Используется правильный набор требований, соответствующий плану управления атрибутами.
Для всех типов требований были заданы атрибуты, позволяющие отслеживать следующую информацию:

Список параметров:

  • Состояние мониторинга
  • Польза
  • Обоснование
  • Трудоемкость реализации
  • Типы и величины рисков, связанных с реализацией
    • Риски для графика
    • Риски для ресурсов
    • Технические риски
  • Стабильность требования
  • Целевой выпуск
  • Назначение
  • Мнение маркетинга
  • Мнение разработчиков
  • Хронология изменения
  • Расположение
  • Причины изменения
  • Убедительность
Заданы все параметры прослеживаемости в соответствии с планом управления требованиями.
