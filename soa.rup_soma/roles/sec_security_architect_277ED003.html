Роль: Security Architect

    Основное описание

    A person in this role leads a group of developers in the task of the development of a security architecture. A security architecture is a representation of the best practices of the security community. It often includes the selection of preferred security patterns for system design (security design patterns), the documentation of technical policy (policy design patterns) concerning security implementation (security implementation patterns) and security architectural patterns reflecting the key technical decisions that constrain the overall security implementation for the project.
