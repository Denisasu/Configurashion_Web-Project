Оглавление 

Модуль RUP для J2EE™ версии 1.4  содержит необходимые компоненты и инструкции по выполнению операций RUP в проекте разработки J2EE.

Модуль RUP для J2EE версии 1.4 также содержит памятки по инструменту, где описаны способы выполнения операций J2EE с помощью инструментов Rational.

Дополнительная информация о базовых концепциях J2EE приведена в разделе   Концепция: Обзор J2EE (Java 2 Platform Enterprise Edition).

Дополнительная информация о взаимодействии J2EE и RUP приведена в разделе  Концепция: Работа с J2EE в RUP.
