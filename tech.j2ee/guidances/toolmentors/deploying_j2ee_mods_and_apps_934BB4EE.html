В этой памятке по инструменту описано развертывание автономных модулей J2EE и приложений J2EE в Rational XDE.
Инструмент: Rational XDE Developer
Основное описание

Назначение

В этой памятке по инструменту описано развертывание автономных Модулей J2EEs и Приложений J2EE с помощью Rational XDE. Процедура развертывания зависит от приложения и среды и обычно требует применения инструментов вендора. В этой памятке по инструменту описаны способы установки, настройки и запуска приложения с помощью Rational XDE. Дополнительная информация приведена в разделе Рекомендация: Развертывание модулей и приложений J2EE.

Обзор

XDE предоставляет средства автоматической настройки модулей и приложений J2EE, которые подлежат развертыванию в определенной среде.

В этой памятке по инструменту предполагается, что архивы и их модели развертывания, содержащие XDE, уже созданы, и что архивы уже привязаны к узлам. Дополнительная информация приведена в разделе Памятка по инструменту: Компоновка модулей и приложений J2EE с помощью Rational XDE Developer - Java Platform Edition.

В этой памятке по инструменту предусмотрены следующие шаги:

  • Развертывание приложения J2EE
  • Настройка приложения J2EE
  • Запуск приложения J2EE

Развертывание приложения J2EE

Некоторые виды настройки, выполняемой до установки (например, настройку файлов описания), можно выполнять в моделях развертывания XDE (либо путем создания новых моделей, оставляя исходные модели без изменений, либо редактируя исходные модели). Дополнительная информация об изменении свойств файлов описания с помощью моделей развертывания XDE приведена в разделе Памятка по инструменту: Компоновка модулей и приложений J2EE с помощью Rational XDE Developer - Java Platform Edition.

После настройки моделей развертывания можно развернуть приложение из модели развертывания EAR. Можно развернуть только один файл EAR или все файлы EAR, привязанные к конкретному узлу. Дополнительная информация приведена в разделе Развертывание приложений Java .

Файл описания или архив можно экспортировать из любой модели развертывания. Экспортирование файла описания применяется для отладки, а также для определения связи между моделью и файлом описания. Экспортирование архивов позволяет развертывать автономные архивы и использовать инструменты развертывания серверов приложений, не поддерживаемые средой XDE. Дополнительная информация приведена в разделе Развертывание приложений Java . См. также раздел Создание файлов описания EJB .

Настройка приложения J2EE

Для правильной работы приложения может потребоваться дополнительная настройка сервера приложений, например, связывание базы данных и настройка доступа к общим библиотекам, которые не в входят в EAR. Эти операции можно выполнить с помощью административной консоли сервера.

Запуск приложения J2EE

После развертывания приложения выполните инструкции вендора сервера по запуску приложения. Во многих случаях серверы, работающие в режиме отладки и тестирования, автоматически идентифицируют новое приложение и немедленно принимают его для тестирования.
