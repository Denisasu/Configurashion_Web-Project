Отчет: Test Survey
The test survey provides information on the structure and contents of the test implementation.
Взаимосвязи
Связанные элементы
  • Комплект тестов
Основное описание
  1. Introduction

  2. Briefly summarizes the scope of the tests.

  3. Overview of Test Suites

  4. Lists the hierarchy of Test Suites including a brief description for each test suite, and shows how Test Suites depends on or utilize the services of other Test Suites.

  5. Test Suite Outlines

  6. For each Test Suite, enumerates a list of Test Scripts each Test Suite uses and-if the test script names are not sufficient-a brief description of each Test Script.

  7. Test Suite Traceability

  8. Optionally, the report may also show traceability references to which Test Motivator(s) and/ or Test Case(s) each Test Suite addresses.
