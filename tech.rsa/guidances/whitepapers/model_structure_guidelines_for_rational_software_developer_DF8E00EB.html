Информационный бюллетень: Model Structure Guidelines for RSx
This paper presents a set of guidelines on how to structure RSx (RSA, RSM and RSD) models.

Abstract

This document provides you with a set of guidelines on how to structure RSA models. These guidelines are an ideal place to start when deciding on the structure of your models.
