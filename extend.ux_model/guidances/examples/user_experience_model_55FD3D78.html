An example of a User-Experience Model can be found on the developerWorks®: Rational® Web site. Just search for "PCOA".

The example is a User-Experience Model for an online auction application called "Pearl Circle Online Auction" (PCOA).
Note: In addition to the User-Experience Model, PCOA also includes other system models (e.g., Use-Case Model, Design Model, etc.).
